# NVIDIA GPU Data Science Project

## Overview
This project focuses on building a real-world NVIDIA GPU ecosystem dataset and performing data cleaning, exploratory data analysis, feature engineering, and statistical analysis.

## Objectives
- Build a messy real-world dataset
- Clean and preprocess data
- Perform EDA
- Generate business insights
- Create engineered features for ML

## Skills Demonstrated
- Pandas
- NumPy
- Data Cleaning
- EDA
- Visualization
- Statistics
- Feature Engineering
